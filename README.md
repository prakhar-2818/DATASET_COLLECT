# 🏠 Housing Market Intelligence Dashboard

A modern **data visualization platform** built using **Flask and Tableau Public** to analyze housing market trends and property insights through interactive dashboards.

This project presents a **data-driven exploration of housing prices, property characteristics, and market patterns**, enabling users to visually understand how different factors influence housing values.

---

## 🚀 Project Overview

The **Housing Market Intelligence Dashboard** provides an interactive analytics interface where users can explore housing data through **dynamic Tableau dashboards and storytelling visualizations**.

The platform focuses on uncovering relationships between:

* Housing sale prices
* Property features
* Renovation history
* Demographic and location patterns

By combining **Flask-based web development with Tableau’s visualization capabilities**, the project delivers a clean and modern analytics interface for exploring real estate insights.

---

## ✨ Key Features

* 📊 **Interactive Tableau Dashboards**
  Explore dynamic charts and visualizations powered by Tableau Public.

* 📈 **Housing Price Trend Analysis**
  Analyze how housing prices change across different features and conditions.

* 🏡 **Property Feature Insights**
  Understand the impact of factors such as renovation status, property age, and property attributes.

* 📖 **Data Storytelling Section**
  A guided visualization narrative explaining patterns and insights in the housing market.

* 🎨 **Modern Responsive UI**
  Built with a sleek SaaS-style interface including animations, hover effects, and glassmorphism elements.

## 🖥 Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/housing-trends-web.git
```

Navigate to the project folder:

```bash
cd housing-trends-web
```

Install Flask if needed:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
